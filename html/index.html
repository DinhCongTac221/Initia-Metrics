Columns with Detailed Metrics

The following columns contain clickable values for detailed metrics:

  • Total Jails
  • Jails [800000 - 2051430]

Block Parsing Information

Note:
Blocks are parsed starting from Height 800,000 to 2,051,430, covering the period until the latest phase ended.
Jails [800000 - 2051430] shows slashing events from block 800,000 onwards.
Total Jails and Voted Proposals include data starting from block 1.

For more information, refer to our GitHub repository and the JSON format.
