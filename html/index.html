Data Parsing Information

This page displays metrics for initial validators on initiation-1 testnet network.

Values in the following columns are clickable to see details of certain metrics: Moniker, Total Jails, Jails [1M> height], Voted Proposals, Val Creation Height

Blocks (Blocks Active, Signed, Missed, Proposed, With Price, Without Price, Uptime) are parsed starting from Height 1 000 000 to 2 051 430 when the latest phase ended

*Jails [1M> height] represents Jail events after Block 1 000 000 while Total Jails and Voted Proposals provide information for the whole network starting from Block 1

For more information, refer to the scripts.js file where parsing and display logic is implemented.
